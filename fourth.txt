
           GIT EXERCISE 

1. Create a folder called learn_git_again. 
Answer: mkdir learn_git_again
2. cd into the learn_git_again folder. 
Answer: cd learn_git_again
3. Create a file called third.txt. 
Answer: touch third.txt
4. Initialize an empty git repository. 
Answer: git init
5. Add third.txt to the staging area. 
Answer: git add third.txt
6. Commit with the message “adding third.txt”.
Answer: git commit -m “adding third.txt”
7. Check out your commit with git log.
Answer: git log
8. Create another file called fourth.txt.
Answer: touch fourth.txt
9. Add fourth.txt to the staging area.
Answer: git add fourth.txt
10. Commit with the message “adding fourth.txt”. 
Answer: git commit -m “adding fourth.txt”
11. Remove the third.txt file.
Answer: rm third.txt
12. Add this change to the staging area.
Answer: git add third.txt
13. Commit with the message “removing third.txt”. 
Answer: git commit -m “removing third.txt”
14. Check out your commits using git log. 
Answer: git log
15. Change your global setting to core.pager=cat 
Answer: git config --global core.pager “cat”
16. Write the command to list all of the global configurations for git on your machine. 
Answer: git config --global --list
